Welcome to PillowTalks 👥

PillowTalks is a safe, anonymous chat platform built using Flask where users can talk freely without revealing their identity. Whether you're looking for deep conversation, casual chat, or just someone to listen — PillowTalks connects you instantly.

✨ Features

  • Anonymous 1-on-1 chat system
  • Interest-based matchmaking
  • Timer-based auto chat session (e.g., 7 mins limit)
  • Flask-SocketIO for real-time chat
  • User authentication system (login/signup/logout)
  • Secure password handling with hashing
  • Flash messages and session management
  • Dashboard for user access

💡 How It Works

Once you're logged in, you can enter the anonymous chat system. Based on your selected interests (e.g., movies, travel, tech, etc.), we match you with a like-minded stranger. The conversation is private, real-time, and disappears after the session ends.

🛠️ Tech Stack

  • Flask – lightweight Python web framework
  • Flask-SocketIO – real-time bi-directional communication
  • Flask-Login – user authentication/session handling
  • SQLite – simple and fast database
  • WTForms – form validation and rendering
  • Jinja2 – templating for dynamic HTML pages

🎯 Why PillowTalks?

In today's fast-paced world, everyone needs a space to talk — freely and without judgment. PillowTalks gives you that space. No names, no profiles, no pressure. Just human connection, pure and simple.
